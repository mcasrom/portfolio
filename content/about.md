## Miguel Castillo

Analista independiente, linuxero convencido y piloto de minidrones.

Construyo dashboards de inteligencia geopolítica con Python y Streamlit, ejecutados en tarjetas ARM Odroid bajo DietPi — sin cloud corporativo, sin dependencias opacas, sin costes de servidor.

**El proyecto** nació de una pregunta simple: ¿se puede hacer análisis de datos serio con hardware minimalista y software libre? La respuesta es sí, y este hub es la demostración.

---

### Stack

- **Hardware** — Odroid XU4 + laptop Lubuntu
- **SO** — DietPi / Ubuntu
- **Lenguajes** — Python, Bash, Emacs Lisp
- **Visualización** — Streamlit, Folium, Plotly
- **Web** — Hugo, ox-hugo, GitHub Pages
- **Automatización** — cron, GitHub Actions
- **Editor** — Emacs + Org-mode + PlantUML
